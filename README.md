# N/Back

A modern, privacy-first dual n-back trainer for the web. Watch positions, listen to letters, and identify matches from *n* steps earlier.

## Features

- Dual 1–5 back training
- Keyboard and touch controls
- Spoken audio cues using the browser speech engine
- Local-only progress history
- Responsive, accessible interface
- No accounts, ads, analytics, or tracking

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Import this repository into Vercel. The framework preset is Vite and the default build settings work as-is.
